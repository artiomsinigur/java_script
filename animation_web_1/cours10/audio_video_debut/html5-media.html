Eveniments clavier

onkeypress: lorsque une touche est activée

onkeydown: lorsque une touche est enfoncée

onkeyup: lorsque une touche est relevée

Quleques propriétés

keyCode: code numerique de la touche du clavier

altKey: true si la touche LAT est activé

ctrlKey: true si la touche CTRL est activé
